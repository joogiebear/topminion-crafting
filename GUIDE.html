ğŸ”§ Configuring Minion IDs

Important: The minion_id must exactly match the ID in your TopMinion configs!

  1. 1.

    Go to your TopMinion configs folder:

    plugins/TopMinion/MinionConfigs/[type]/
  2. 2.

    Open a minion config file (e.g., miner-config-example.yml)

  3. 3.

    Find the minion_id line at the top:

    minion_id: MINER
  4. 4.

    Copy that exact ID and use it in your recipe config:

    minion_id: "MINER" # Must match exactly!

ğŸ“ Recipe Configuration

Shaped Recipe Example

miner_basic:
  enabled: true                    # Enable/disable this recipe
  minion_id: "MINER"               # Must match TopMinion config ID
  minion_level: 1                  # Always 1 for crafting
  recipe_type: "SHAPED"            # Recipe requires specific pattern
  shape:
    - "CCC"                        # Top row
    - "CPC"                        # Middle row
    - "CCC"                        # Bottom row
  ingredients:
    C: COBBLESTONE                 # C = Cobblestone
    P: DIAMOND_PICKAXE             # P = Diamond Pickaxe
  permission: "custom.permission"  # OPTIONAL - can be omitted!

Shapeless Recipe Example

autosell_basic:
  enabled: true
  upgrade_id: "autosell"
  recipe_type: "SHAPELESS"         # Items can go anywhere
  ingredients:
    - EMERALD
    - CHEST
    - GOLD_INGOT
  # No permission specified - uses default!

ğŸ”§ Troubleshooting

âŒ Recipes Not Working / Not Appearing

Possible Causes:

  • TopMinion is not installed or not enabled
  • The minion_id or upgrade_id doesn't match your TopMinion config
  • Recipe is disabled (enabled: false)
  • Invalid material names in ingredients

Solutions:

  • Run /plugins and verify TopMinion shows as green
  • Check TopMinion config files and match the IDs exactly
  • Enable debug mode in config (settings.debug: true) and check console
  • Run /topminioncrafting reload after config changes

âŒ Getting Wrong Item When Crafting

Possible Causes:

  • The minion_id or upgrade_id is incorrect
  • TopMinion API returned null (check console for errors)

Solutions:

  • Verify the ID matches exactly in TopMinion configs (case-sensitive!)
  • Check server console for error messages
  • Make sure TopMinion has the minion/upgrade configured properly

âŒ Permission Denied When Crafting

Possible Causes:

  • Player doesn't have the required permission
  • Permission plugin not configured correctly

Solutions:

  • Check what permission the recipe requires in config.yml
  • Grant the permission using your permission plugin (e.g., LuckPerms)
  • Default permissions are topminioncrafting.craft.minion and topminioncrafting.craft.upgrade
  • Remove custom permission line from recipe to use defaults

âŒ Plugin Won't Load

Possible Causes:

  • TopMinion is not installed
  • Wrong server version (requires 1.19+)
  • Java version too old (requires Java 17+)

Solutions:

  • Install TopMinion v3 first
  • Update your server to 1.19 or higher
  • Update Java to version 17 or higher
  • Check console for specific error messages

â“ FAQ

Can I craft higher level minions?

No, this addon only supports crafting level 1 minions. Players can then upgrade them using TopMinion's normal upgrade system. This is by design to balance progression.

Can I craft skins?

No, SKIN type upgrades are not craftable. This is intentional - skins are meant to be special/rare items obtained through other means (shops, crates, events, etc.).

How do I change recipe ingredients?

Edit the config.yml file and change the materials under ingredients. Make sure to use valid Minecraft material names (e.g., DIAMOND, IRON_INGOT, etc.). After editing, run /topminioncrafting reload.

Can I add more minion types?

Yes! If you create new minion configs in TopMinion, you can add recipes for them in this plugin's config. Just follow the same format and use the correct minion_id.

What's the difference between SHAPED and SHAPELESS recipes?

SHAPED: Items must be placed in specific positions (like a vanilla crafting recipe). Defined using the shape parameter.

SHAPELESS: Items can be placed anywhere in the crafting grid. Just list the ingredients.

Do I need to restart after config changes?

No! Just run /topminioncrafting reload (or /tmc reload) to reload the config and recipes without restarting your server.

Can I make recipes require custom permissions?

Yes! Each recipe can have an optional permission field. If specified, only players with that permission can craft the item. If omitted, it uses the default permissions (topminioncrafting.craft.minion or topminioncrafting.craft.upgrade).
