Commit:
-------
git commit -m "Initial release - TopMinion-Crafting v1.0.0"

Create Repository on GitHub:
-----------------------------
1. Go to github.com
2. Create new repository
3. Name it: TopMinion-Crafting
4. Don't initialize with README (you already have one)

Push to GitHub:
---------------
git remote add origin https://github.com/YOUR_USERNAME/TopMinion-Crafting.git
git branch -M main
git push -u origin main

⚠️ IMPORTANT NOTES:
========================================

1. TopMinion.jar should NOT be committed
   - It's proprietary/paid software
   - Users must provide their own copy
   - lib/README.md explains where to get it

2. Built JARs should NOT be committed
   - Users should build from source
   - Or download from GitHub Releases

3. GitHub Releases
   - After pushing, you can create a Release
   - Upload TopMinion-Crafting-1.0.0.jar there
   - Users can download pre-built JARs from Releases

4. License
   - Add a LICENSE file if you want
   - MIT, GPL, Apache 2.0, etc.
   - Clarify TopMinion is required but separate
